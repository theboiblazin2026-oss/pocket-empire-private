# Zero to Hero: Interactive Curriculum App

This is your interactive guide to learning web development. It was built using the exact technologies you will learn in Phase 3!

## How to Run
1.  Open your terminal (in VS Code or separate).
2.  Navigate to this folder: `cd interactive-curriculum-app`
3.  Install dependencies (if you haven't): `npm install`
4.  Start the app: `npm run dev`
5.  Open the link shown (usually `http://localhost:5173`) in your browser.

## Features
*   **Progress Tracking**: Your progress is saved automatically.
*   **Curriculum**: All phases from the text guide are included.
*   **Confetti**: Celebrate your wins!
