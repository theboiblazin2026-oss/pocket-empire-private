# E-commerce Skill

## Checkout Optimization

### Friction Points
- Account required
- Too many fields
- Hidden fees
- Limited payment options
- Slow page load

### Best Practices
- Guest checkout option
- Progress indicator
- Address autocomplete
- Multiple payment methods
- Trust badges
- Clear return policy

## Cart Abandonment

**Average rate: 70%**

### Recovery Tactics
1. Exit-intent popup (offer)
2. Email sequence:
   - 1 hour: Reminder
   - 24 hours: Urgency
   - 72 hours: Discount

### Email Template
```
Subject: You left something behind...

Hi {name},

You left items in your cart! Complete your order and get 10% off with code SAVE10.

[Complete My Order]

Your cart:
- {item} - ${price}
```

## Inventory Management

| Method | Description |
|--------|-------------|
| FIFO | First in, first out |
| LIFO | Last in, first out |
| JIT | Just-in-time ordering |

## Key Metrics

| Metric | Formula |
|--------|---------|
| Conversion Rate | Orders / Visitors |
| AOV | Revenue / Orders |
| LTV | AOV × Purchase Frequency × Lifespan |
| CAC | Marketing Spend / New Customers |

## When to Apply
Use when optimizing online stores, reducing cart abandonment, or managing inventory.
