# IP/Patents Skill

## IP Types

| Type | Protects | Duration | Symbol |
|------|----------|----------|--------|
| Copyright | Creative works | Life + 70 years | © |
| Trademark | Brand identity | Indefinite (if renewed) | ™ ® |
| Patent | Inventions | 20 years | |
| Trade Secret | Confidential info | As long as secret | |

## Copyright

**Automatic** upon creation for:
- Code, software
- Written content
- Images, videos
- Music

**Registration benefits:**
- Sue for infringement
- Statutory damages
- Public record

## Trademark

**Can protect:**
- Names
- Logos
- Slogans
- Sounds, colors (harder)

**Process:**
1. Search USPTO database
2. File application
3. Examination (6-12 months)
4. Publication for opposition
5. Registration

## Provisional Patent

**Benefits:**
- "Patent Pending" status
- 12-month priority date
- Lower cost ($300-$1,500)
- Time to validate idea

**Must include:**
- Detailed description
- Drawings
- Cover sheet

## When to Apply
Use when protecting intellectual property, filing patents, or licensing IP.
