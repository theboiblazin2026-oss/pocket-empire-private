# Meditation Guide Skill

## Basic Breath Meditation

1. Sit comfortably, spine straight
2. Close eyes or soft gaze
3. Notice breath (don't control it)
4. When mind wanders, return to breath
5. Start with 5 minutes, build up

**The goal isn't to stop thinking. It's to notice when you're distracted and return.**

## Breathing Techniques

### Box Breathing (4-4-4-4)
```
Inhale: 4 seconds
Hold: 4 seconds
Exhale: 4 seconds
Hold: 4 seconds
```
**Use for:** Calming, focus, before stressful events

### 4-7-8 Breathing
```
Inhale: 4 seconds
Hold: 7 seconds
Exhale: 8 seconds
```
**Use for:** Sleep, deep relaxation

### Physiological Sigh
```
Double inhale (nose)
Long exhale (mouth)
```
**Use for:** Instant calm (1-3 breaths)

## Meditation Types

| Type | Focus | Best For |
|------|-------|----------|
| Mindfulness | Present moment awareness | Daily practice |
| Body scan | Tension in body | Relaxation |
| Loving-kindness | Compassion phrases | Emotional healing |
| Visualization | Mental imagery | Goals, performance |
| Mantra | Repeated phrase | Concentration |

## Body Scan Script

```
Notice your feet...
Move up to calves...
Notice thighs...
Awareness of hips...
Feel your stomach...
Notice chest rising/falling...
Relax shoulders...
Release jaw...
Soften forehead...
```

## Building a Practice

- Same time daily (habit stacking)
- Start with 5 minutes
- Use app: Headspace, Calm, Insight Timer
- Track streaks
- Don't judge "bad" sessions

## When to Apply
Use when stressed, building mindfulness practice, or improving focus.
