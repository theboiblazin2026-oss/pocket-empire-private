# LinkedIn Growth Skill

## Profile Optimization

### Headline Formula
```
[Role] | Helping [audience] achieve [outcome] | [Credibility]

Example:
Trucking Automation Expert | Helping carriers save 10+ hours/week | Built 50+ automation systems
```

### About Section
- First 3 lines = hook (visible without "see more")
- Tell your story
- Show results
- End with CTA

### Profile Checklist
- [ ] Professional photo
- [ ] Custom banner
- [ ] Headline with value prop
- [ ] About section complete
- [ ] Featured content
- [ ] Experience with results
- [ ] Skills (get endorsements)

## Content Strategy

### Post Types That Work
| Type | Engagement |
|------|------------|
| Personal story | High |
| Carousel | High |
| How-to/tips | Medium |
| Industry news + take | Medium |
| Text-only insights | Medium |

### Posting Formula
```
Line 1: Hook (stop the scroll)
Line 2: (blank line)
Line 3-7: Story/value
Line 8: (blank line)
Line 9: CTA or question
```

### Best Times to Post
- Tuesday-Thursday
- 8-10 AM
- 12 PM
- 5-6 PM

## Lead Generation

### Outreach Template
```
Hi [Name],

Saw your post about [topic] - great insight on [specific point].

I help [their industry] with [your value]. Would love to connect and learn more about your work.

[Your name]
```

### Connection Request (300 char limit)
```
Hi [Name], I'm connecting with [industry] professionals. Noticed you're [observation]. Would love to connect!
```

## When to Apply
Use when optimizing LinkedIn profile, creating content, or generating leads.
