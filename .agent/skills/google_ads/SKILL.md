# Google Ads Skill

## Campaign Types

| Type | Best For |
|------|----------|
| Search | High-intent keywords |
| Display | Brand awareness |
| Shopping | E-commerce products |
| Video | YouTube engagement |
| Performance Max | Automated cross-channel |

## Search Ad Structure

```
Headline 1 (30 chars) | Headline 2 (30 chars) | Headline 3 (30 chars)
Description 1 (90 chars)
Description 2 (90 chars)
Display URL with paths
```

## Keyword Match Types

| Type | Symbol | Example |
|------|--------|---------|
| Broad | none | running shoes |
| Phrase | "..." | "running shoes" |
| Exact | [...] | [running shoes] |

## Quality Score Factors

1. Expected CTR
2. Ad relevance
3. Landing page experience

**Higher score = Lower CPC**

## Bidding Strategies

| Strategy | Goal |
|----------|------|
| Manual CPC | Full control |
| Maximize Clicks | Traffic |
| Maximize Conversions | Conversions |
| Target CPA | Cost per acquisition |
| Target ROAS | Return on ad spend |

## Key Metrics

- CTR (Click-through rate)
- CPC (Cost per click)
- Conversion rate
- CPA (Cost per acquisition)
- ROAS (Return on ad spend)

## When to Apply
Use when setting up Google Ads campaigns or optimizing PPC performance.
