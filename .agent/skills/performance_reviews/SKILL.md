# Performance Reviews Skill

## Review Types

| Type | Frequency | Purpose |
|------|-----------|---------|
| 1:1 | Weekly | Ongoing feedback |
| Quarterly | Every 3 months | Goal check-in |
| Annual | Yearly | Formal review |
| 360 | Annual | Multi-source feedback |

## OKR Framework

**Objective:** Qualitative goal
**Key Results:** Measurable outcomes (3-5 per objective)

```
Objective: Launch successful product
KR1: Ship v1.0 by March 31
KR2: Acquire 1,000 beta users
KR3: Achieve 4.0+ app store rating
```

## Feedback Framework (SBI)

- **Situation:** "In yesterday's meeting..."
- **Behavior:** "You interrupted the presenter..."
- **Impact:** "Which made them lose their train of thought."

## Self-Review Prompts

1. What accomplishments are you most proud of?
2. What challenges did you face?
3. What skills do you want to develop?
4. What do you need from your manager?

## Performance Improvement Plan (PIP)

| Section | Content |
|---------|---------|
| Issue | Specific performance gap |
| Expectations | Clear success criteria |
| Support | Resources provided |
| Timeline | 30-60-90 day milestones |
| Consequences | What happens if not met |

## When to Apply
Use when conducting reviews, writing feedback, or managing performance issues.
