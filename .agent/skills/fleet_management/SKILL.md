# Fleet Management Skill

## Growth Path

```
Owner-Operator (1 truck)
        ↓
Small Fleet (2-5 trucks)
        ↓
Medium Fleet (6-20 trucks)
        ↓
Large Fleet (20+ trucks)
```

## Adding Your First Driver

### Driver Types

| Type | Pros | Cons |
|------|------|------|
| Company Driver | You control everything | Payroll, taxes, liability |
| Lease-to-Own | Driver pays for truck | Less control |
| Owner-Op under your authority | Low overhead | They can leave |

### Pay Structure
- **Percentage:** 25-30% of gross
- **Per mile:** $0.50-0.65/mile
- **Salary + bonus:** Base + performance

## Fleet Economics

```
Revenue per truck: $15,000/month
- Fuel: $4,000
- Driver: $4,500
- Insurance: $1,500
- Maintenance: $500
- Dispatch fee: $750
- Misc: $500
= Profit: $3,250/truck/month

5 trucks = $16,250/month profit
```

## Key Metrics

| Metric | Target |
|--------|--------|
| Revenue per mile | $2.50+ |
| Empty miles | < 10% |
| Fuel cost/mile | < $0.50 |
| Maintenance/mile | < $0.10 |
| Driver turnover | < 30%/year |

## Systems Needed

- **Dispatch:** TruckLogs, Axon, Rose Rocket
- **ELD:** KeepTruckin, Samsara
- **Accounting:** QuickBooks + trucker integration
- **Insurance:** Progressive, OOIDA, Great West

## Hiring Drivers

Red flags:
- Job hopping (6+ jobs in 3 years)
- Accidents on MVR
- Multiple speeding tickets
- Can't provide references

Green flags:
- 2+ years experience
- Clean MVR
- Stable work history
- CDL-A with current medical

## When to Apply
Use when planning fleet expansion, hiring drivers, or scaling operations.
