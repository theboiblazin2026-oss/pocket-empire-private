# Onboarding Skill

## First 90 Days Framework

### Week 1: Orientation
- [ ] Equipment setup
- [ ] Account access
- [ ] Meet the team
- [ ] Company overview
- [ ] Role expectations

### Days 8-30: Learning
- [ ] Product deep-dive
- [ ] Shadow team members
- [ ] Review documentation
- [ ] First small task
- [ ] 30-day check-in

### Days 31-60: Contributing
- [ ] Own small projects
- [ ] Regular 1:1s
- [ ] Process feedback
- [ ] 60-day check-in

### Days 61-90: Accelerating
- [ ] Independent work
- [ ] Cross-team collaboration
- [ ] Identify improvements
- [ ] 90-day review

## Onboarding Checklist

### Before Day 1
- [ ] Send welcome email
- [ ] Ship equipment
- [ ] Create accounts
- [ ] Assign buddy/mentor
- [ ] Schedule first week meetings

### Day 1
- [ ] Welcome meeting
- [ ] Office/team tour
- [ ] Equipment setup
- [ ] HR paperwork
- [ ] Lunch with team

## Documentation to Prepare

- Employee handbook
- Team wiki/docs
- Tool guides
- Org chart
- FAQs

## When to Apply
Use when onboarding new hires or creating onboarding programs.
