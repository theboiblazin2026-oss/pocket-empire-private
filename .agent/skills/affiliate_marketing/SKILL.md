# Affiliate Marketing Skill

## How It Works

```
You share link → Someone buys → You earn commission
```

## Content That Converts

### Best Formats
1. **"Best X for Y"** - "Best laptop for truckers"
2. **Reviews** - Honest product reviews
3. **Comparisons** - "X vs Y"
4. **Tutorials** - "How to use X"
5. **Resource pages** - Curated tools lists

### Link Placement
- Above the fold
- Within content (natural)
- Comparison tables
- Call-out boxes

## Disclosure Requirements

**FTC requires disclosure:**
> "This post contains affiliate links. I may earn a commission at no extra cost to you."

Always be transparent.

## Tracking & Analytics

Track:
- Click-through rate
- Conversion rate
- Earnings per click (EPC)
- Top performing content

## Getting Started

1. Choose a niche you know
2. Create valuable content
3. Join relevant programs
4. Add links naturally
5. Scale what works

## When to Apply
Use when monetizing content, choosing programs, or optimizing conversions.
