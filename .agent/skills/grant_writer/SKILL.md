# Grant Writer Skill

## Proposal Structure

### 1. Executive Summary (1 page)
- Organization overview
- Problem statement
- Proposed solution
- Funding request
- Expected outcomes

### 2. Statement of Need
- Data-backed problem description
- Who is affected
- Why it matters now
- Gap in current solutions

### 3. Project Description
- Goals and objectives (SMART)
- Methods and activities
- Timeline
- Staffing plan

### 4. Evaluation Plan
- How you'll measure success
- Metrics and KPIs
- Reporting schedule

### 5. Budget
- Personnel costs
- Direct costs (supplies, travel)
- Indirect costs (overhead)
- Budget narrative

### 6. Organizational Capacity
- Mission and history
- Past successes
- Staff qualifications
- Financial stability

## Funding Sources

| Type | Examples |
|------|----------|
| Federal | NIH, NSF, SBIR, DOE |
| Foundation | Gates, Ford, MacArthur |
| Corporate | Google.org, Walmart Foundation |
| State/Local | Economic development, arts councils |

## SBIR/STTR Basics

| Phase | Award | Duration |
|-------|-------|----------|
| Phase I | $50K - $275K | 6-12 months |
| Phase II | $750K - $1.5M | 2 years |
| Phase III | Private funding | Commercialization |

## Common Mistakes

- Not following guidelines exactly
- Weak needs statement
- Vague objectives
- Budget doesn't match narrative
- Missing required attachments
- Last-minute submission

## When to Apply
Use when writing grant proposals, researching funding, or developing nonprofit programs.
