# DOT Compliance Advanced Skill

## CSA Scores (BASICS)

| Category | Weight | What Triggers |
|----------|--------|---------------|
| Unsafe Driving | High | Speeding, reckless |
| HOS Compliance | High | Log violations |
| Driver Fitness | Medium | Medical cert, CDL |
| Controlled Substances | High | Drug/alcohol |
| Vehicle Maintenance | High | OOS violations |
| Crash Indicator | High | DOT-reportable crashes |
| Hazmat | High (if applicable) | Hazmat violations |

### Score Thresholds (Intervention)

| BASIC | Threshold |
|-------|-----------|
| Unsafe Driving | 65% |
| HOS | 65% |
| Crash | 65% |
| Vehicle Maint. | 80% |

## Surviving an Audit

### What FMCSA Looks For
1. Driver qualification files (DQ files)
2. Drug testing records (clearinghouse)
3. Hours of Service logs (6 months)
4. Vehicle maintenance records
5. Insurance certificates
6. Accident register

### Audit Prep Checklist
- [ ] DQ files complete for all drivers
- [ ] Pre/post trip inspections documented
- [ ] DVIR forms organized
- [ ] Annual inspections current
- [ ] DOT med cards not expired
- [ ] Clearinghouse queries done

## Common Violations (Avoid These)

| Violation | Severity |
|-----------|----------|
| No medical certificate | High |
| False log entries | Critical |
| Driving beyond hours | High |
| No fire extinguisher | Medium |
| Inoperative lights | Medium |
| No triangle kit | Medium |

## Clearinghouse

**Query all drivers annually!**
- Full query: Requires driver consent
- Limited query: No consent needed
- Check before hiring

## When to Apply
Use when preparing for audits, managing CSA scores, or ensuring compliance.
