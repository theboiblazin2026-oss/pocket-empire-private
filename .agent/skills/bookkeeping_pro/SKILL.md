# Bookkeeping Pro Skill

## Chart of Accounts

| Type | Number Range | Examples |
|------|--------------|----------|
| Assets | 1000-1999 | Cash, AR, Equipment |
| Liabilities | 2000-2999 | AP, Loans, Credit Cards |
| Equity | 3000-3999 | Owner's Equity, Retained Earnings |
| Revenue | 4000-4999 | Sales, Service Income |
| Expenses | 5000-5999 | Rent, Utilities, Supplies |

## Double-Entry Basics

Every transaction has two sides:

| Debit | Credit |
|-------|--------|
| Increase Assets | Decrease Assets |
| Decrease Liabilities | Increase Liabilities |
| Decrease Equity | Increase Equity |
| Increase Expenses | Decrease Expenses |
| Decrease Revenue | Increase Revenue |

**Example:** Paid $500 for rent
- Debit: Rent Expense +$500
- Credit: Cash -$500

## Monthly Close Checklist

- [ ] Reconcile all bank accounts
- [ ] Reconcile credit card statements
- [ ] Review outstanding invoices (AR)
- [ ] Review unpaid bills (AP)
- [ ] Record depreciation
- [ ] Accrue payroll if applicable
- [ ] Review P&L for anomalies
- [ ] Generate financial statements

## Financial Statements

### Income Statement (P&L)
Revenue - Expenses = Net Income

### Balance Sheet
Assets = Liabilities + Equity

### Cash Flow Statement
Operating + Investing + Financing = Net Cash Change

## Key Metrics

| Metric | Formula |
|--------|---------|
| Gross Margin | (Revenue - COGS) / Revenue |
| Net Margin | Net Income / Revenue |
| Current Ratio | Current Assets / Current Liabilities |
| Quick Ratio | (Current Assets - Inventory) / Current Liabilities |
